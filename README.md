把英语课堂“戴”在手上，学习也能这么酷！本项目为小米手环用户提供离线单词库与掌上查询功能，让学习更碎片化、便捷化。

## 核心亮点

- **海量词库**：覆盖小学、初中、高中、四六级、雅思、托福等，单词量高达 10w+，从课本到考试全面覆盖。
- **离线学习**：支持断开手机蓝牙后仍可使用，适合学校、图书馆、通勤等场景随时查词背词。
- **机型适配**：已适配小米手环10、10 NFC、9 Pro，抬手即学，操作便捷。

## 适用人群

- **中小学生**：背单词更有趣，利用碎片时间轻松记忆。
- **住校党 / 无法常带手机用户**：无需手机即可查词，学习效率提升。
